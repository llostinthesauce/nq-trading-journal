# nq-trading-journal

iOS trading journal app for tracking trades

## Overview

A native iOS application built with SwiftUI for tracking and analyzing trading activity. Features include trade entry, performance analytics, charts, and journal notes. Designed for active traders who want to maintain detailed records of their trading decisions and outcomes.

## Tech Stack

- Swift
- SwiftUI
- Core Data
- iOS SDK
- Xcode

## Installation

```bash
# Clone the repository
git clone https://github.com/llostinthesauce/nq-trading-journal.git
cd nq-trading-journal

# Open in Xcode
open nq.xcodeproj
```

## Usage

1. Open the project in Xcode
2. Select a simulator or connected iOS device
3. Build and run (Cmd+R)

Features include:
- Add and track individual trades
- View performance statistics
- Review trade history
- Add notes and observations
